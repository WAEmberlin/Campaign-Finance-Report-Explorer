# Changelog

## 1.0.0 — 2026-07-18

### Added
- Browser-only SPA with Bootstrap 5 shell, dark Kansas-blue theme, PWA support
- IndexedDB storage with schema migrations and statewide reference data
- Dual ingest: drag-and-drop PDFs/JSON and Refresh From Kansas catalog
- PDF text parser (PDF.js) plus structured JSON report import
- Analytics engine, PAC/vendor classification, Fuse.js search, deterministic insights
- District dashboard, Campaign Intelligence Center, donor/expense explorers
- Chart.js visualizations with PNG export
- D3 relationship explorer and Leaflet district map
- CSV/JSON/clipboard/share-URL exports
- GitHub Pages deploy workflow and kansas-sync catalog workflow
