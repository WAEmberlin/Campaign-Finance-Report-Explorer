Help

How to import and explore Kansas campaign finance data.

  1. Set Election Cycle, Office, and District filters (defaults: 2026 / House / 70).
  2. Click Refresh From Kansas to clear local imports and reload mirrored catalog reports, or drop official PDFs.
  3. Explore dashboards, search, relationship graph, and maps. Exports are available as CSV / JSON / PNG.

Live downloads from kansas.gov are blocked by browser CORS. The kansas-sync GitHub Action refreshes data/filings-index.json; mirrored report JSON belongs under data/reports/.

About

Open-source civic transparency platform for Kansas campaign finance.

Kansas Campaign Finance Explorer runs entirely in the browser — no accounts, no backend, no API keys. Data is normalized from Receipts & Expenditures reports into structured entities for analytics.
